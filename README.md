## Gym Management System with Multiple Identities

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1649556afe5741b0a4d4b3410b9c94d4/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Store Manager, Front Desk, On-site Staff, Coach, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Venue Information
5. Communication Forum
6. Fitness Courses
7. Announcement Notifications
8. Personal Center
9. Venue Favorites
10. Venue Comments
11. Venue Booking Application
12. Course Favorites
13. Course Comments
14. Course Orders

**Store Manager Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Store Manager Management
6. Coach Management
7. Front Desk Management
8. On-site Staff Management
9. Fitness Course Management
10. Course Comment Management
11. Course Favorite Management
12. Course Order Management
13. Venue Information Management
14. Venue Comment Management
15. Venue Favorite Management
16. Venue Booking Application Management
17. Fitness Equipment Management
18. Inventory Management
19. Inventory Detail Management
20. Communication Forum Management
21. Announcement Notification Management


**Front Desk Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. User Management
6. Fee Management
7. Communication Forum Management
8. Announcement Notification Management

**On-site Staff Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Fitness Equipment Management
6. Inventory Management
7. Inventory Detail Management


**Coach Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Course Order Management







**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Store Manager Management
5. Coach Management
6. Front Desk Management
7. On-site Staff Management
8. User Management
9. Fitness Course Management
10. Course Comment Management
11. Course Favorite Management
12. Course Order Management
13. Venue Information Management
14. Venue Comment Management
15. Venue Favorite Management
16. Venue Booking Application Management
17. Fitness Equipment Management
18. Inventory Management, Inventory Detail Management
19. Fee Management
20. Basic Data Management (Venue Type Management, Fitness Course Type Management, Fitness Equipment Type Management, Announcement Type Management, Fee Type Management)
21. Communication Forum Management
22. Announcement Notification Management
23. Carousel Management
